# 🐦 Twiffty

Twiffty is a Flappy Bird–style arcade game built using Flutter.  
Tap to keep the character flying and avoid obstacles for as long as possible.

---

## 🎮 Gameplay

- Tap to fly upward
- Avoid obstacles and the ground
- Score increases when passing obstacles
- Collision ends the game

---

## ✨ Features

- Classic Flappy Bird–inspired gameplay
- Smooth animations
- Simple and responsive UI
- Real-time scoring system

---

## 🛠️ Built With

- Flutter
- Dart
